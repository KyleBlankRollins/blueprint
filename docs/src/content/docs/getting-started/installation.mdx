Blueprint is published to npm as `@krollins/blueprint`. It works in any project that supports ES modules and web components.

## Install

```bash
npm install @krollins/blueprint
```

## Importing Components

### Import All Components

Import the full library to register every component at once:

```javascript
import '@krollins/blueprint';
```

This registers all `bp-*` custom elements and includes the theme CSS (design tokens, fonts, colors).

### Import Individual Components (Recommended)

For smaller bundles, import only the components you need:

```javascript
import '@krollins/blueprint/button';
import '@krollins/blueprint/input';
import '@krollins/blueprint/card';
```

Each sub-path import registers that component's custom element as a side effect. You still need to import the theme CSS separately when using individual imports:

```javascript
import '@krollins/blueprint/dist/index.css';
```

Without the theme CSS, components render without styling because the `--bp-*` custom properties they reference won't exist.

## Framework Integration

### Astro

For Astro projects, add the `@astrojs/lit` integration:

```bash
npx astro add lit
```

Import the theme CSS in your layout's frontmatter, and component JS in a client-side `<script>` tag:

```astro
---
import '@krollins/blueprint/dist/index.css';
---

<script>
  import '@krollins/blueprint/button';
  import '@krollins/blueprint/card';
  import '@krollins/blueprint/heading';
</script>

<bp-button variant="primary">Click me</bp-button>
```

### React

Blueprint components work directly in React without wrappers:

```jsx
import '@krollins/blueprint/dist/index.css';
import '@krollins/blueprint/button';
import '@krollins/blueprint/input';

function App() {
  return (
    <div>
      <bp-button onClick={() => alert('Clicked!')}>Click me</bp-button>
      <bp-input placeholder="Enter text" />
    </div>
  );
}
```

### Vue

Use Blueprint components like any other HTML element:

```vue
<template>
  <div>
    <bp-button @click="handleClick">Click me</bp-button>
    <bp-input :value="inputValue" @bp-input="handleInput" />
  </div>
</template>

<script setup>
import '@krollins/blueprint/dist/index.css';
import '@krollins/blueprint/button';
import '@krollins/blueprint/input';
</script>
```

### Svelte

Blueprint components work directly in Svelte:

```svelte
<script>
  import '@krollins/blueprint/dist/index.css';
  import '@krollins/blueprint/button';
</script>

<bp-button on:click={() => alert('Clicked!')}>Click me</bp-button>
```

## TypeScript Support

Blueprint includes full TypeScript definitions. The package provides:

- Component class types (`BpButton`, `BpInput`, etc.)
- JSX type augmentations for Astro, React, and Solid.js
- `HTMLElementTagNameMap` declarations for DOM APIs

For JSX environments, types are automatically available when you import the library.
